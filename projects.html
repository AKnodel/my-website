Iris Flower Classification

3 August, 2023

This project employs a TensorFlow neural network to classify Iris flowers by species using petal and sepal measurements. Python libraries like matplotlib, scikit-learn, numpy, and pandas aid in visualization and data manipulation. The Jupyter Notebook 'ml_model.ipynb' outlines the process, from data loading and preprocessing to model training and evaluation. A 'data' folder contains the dataset ('IRIS.csv') and a visualization ('output.png'). The project showcases TensorFlow, scikit-learn, and Jupyter Notebook skills for flower classification.

House Price Prediction

3 August, 2023

This project involves creating a machine learning model using linear regression to predict house prices. Techniques such as scikit-learn, Lasso, L2 regularization, feature engineering, and one-hot encoding are utilized for accuracy. The Jupyter notebook provides step-by-step instructions and code implementation. The 'Housing.csv' dataset is used, and matplotlib is employed for data visualization. Contributions are welcome, and acknowledgments are given to scikit-learn, 'kaggle,' and the open-source community.

Ultimate Checkers

25 June, 2023

This Python checkers game includes an AI opponent utilizing the minimax algorithm with alpha-beta pruning. Users can pick difficulty levels, themes, and play against human or AI. Implemented using 'main.py' or 'Ultimate_Checkers.exe', the AI opponent's depth is based on chosen difficulty. Inspired by Tech With Tim's Checkers AI project.
